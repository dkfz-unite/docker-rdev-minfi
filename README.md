# R-Dev Minfi Docker Image
Docker image for R-Dev with pre-installed Minfi (depends on [rdev-biocmanager](https://github.com/dkfz-unite/docker-rdev-biocmanager)).

**Base image**: ubuntu:latest  
**Packages**: minfi, limma, IlluminaHumanMethylation450kmanifest, IlluminaHumanMethylationEPICmanifest, IlluminaHumanMethylationEPICanno.ilm10b4.hg19
